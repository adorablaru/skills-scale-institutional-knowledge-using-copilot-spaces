# OctoAcme Project Management Docs

Welcome to OctoAcme's project management process documentation. This README gives an overview of how we manage projects and provides links to key living documents.

## OctoAcme Project Management Processes (Summary)

- **Lifecycle:** Initiation → Planning → Execution → Release → Close & Retrospective
- **Principles:** Customer-first, Iterative delivery, Clear ownership, Data-driven
- **Roles:** PM, Product Manager (PdM), Developers, QA, Stakeholders
- **Artifacts:** One-pager/Charter, Backlog, Roadmap, Release notes, Risk Register, Retros
- **Communication:** Weekly syncs, standups, regular stakeholder updates; escalation and risk tracking as needed
